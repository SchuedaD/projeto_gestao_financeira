# Sistema de Gestão Financeira Pessoal

## Descrição

Trata-se do meu primeiro em python para gerenciar finanças pessoais, permitindo o registro de receitas e despesas, categorização de transações e geração de relatórios financeiros.

Além disso, utilizo o notebook para ter um controle dessas finanças com gráficos que vão demonstrar como estão sendo as despesas e receitas. 

## Melhorias em andamento 

- Notebook completo com os melhores dados para controle das finanças
- Base de dados em SQLite
- Melhorias no sistema_financeito.py para utilização por usuários reais

## Como Usar
1. Execute: `python sistema_financeiro.py`
2. Escolha uma opção do menu
3. Suas transações são salvas automaticamente
4. Executar o notebook para análise dos dados que estão em 'transacoes.txt'

## Funcionalidades
- Adicionar receitas e despesas
- Visualizar saldo atual
- Relatórios por categoria
- Persistência em arquivo TXT 

## Tecnologias
- Python 3.x
- Manipulação de arquivos
- Estruturas de dados
